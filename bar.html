Readme2HTML

Readme2HTML is a simple commandline javascript utility which takes a README.md markdown page and converts to an index.html page. This is useful for build scripts where you want to able to just browse a directory and have see the same content as a README.md but don't have a markdown viewer installed (say as a browser extension).

## Dependancies

readm2html uses the nodejs javscript runtime along with some package which are available via the npmjs javascript packaging service.

* nodejs javascript runtime -  [nodejs](https://nodejs.org/en/)  
* npm - package manager javascript packages - [npm](https://docs.npmjs.com/cli/install)
* uses showdown.js for markdown parsing - [showdown.js](https://www.npmjs.com/package/showdown)
* uses bitwrench.js for page formatting duties - [bitwrench](https://www.npmjs.com/package/bitwrench)
